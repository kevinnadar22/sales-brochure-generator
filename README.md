# Sales Brochure Generator

A web application for generating sales brochures using AI.

## Quick Install

```bash
uv sync
uv run main.py
```

## Setup

1. Copy `sample.env` to `.env`
2. Add your OpenAI API key to the `.env` file
3. Run the application using the commands above

## Requirements

- Python 3.13+
- uv package manager
